# Sarah's Corner Website

This is a modern responsive starter website for Sarah's Corner. update.

## Pages
- Home: `index.html`
- About: `about.html`
- Products: `products.html`
- Contact: `contact.html`
- Login / Purchases demo: `login.html`

## Important setup notes
1. The login page is a front-end demo only. Real customer accounts and purchase history require a backend, database, and payment/order system.
2. The contact form uses a placeholder Formspree action. Replace `YOUR_FORM_ID` in `contact.html` with your real Formspree form ID, or connect it to your backend.
3. Product content is currently placeholder content inside `script.js`.
